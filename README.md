# 数据科学基础大作业说明

## 主题相关

这次大作业的主题是有关利用Python进行数据可视化处理的课题

我们将从多个方面来对Python可视化进行描述，举例可以有：

+ 可以利用的库，如matplotlib、Pygal等
+ 可以处理的文件，如csv文件、json文件
+ 可以与其他哪些技术结合，如通过网站API

此外，我们还需要试着去运用更多的方法对案例中的数据进行分析，比如：

+ 假设检验
+ 数据降维
+ 线性回归分析
+ 等等
